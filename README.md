# Install
<pre><code>
git clone https://github.com/skschoo1/skphp
cd skphp
</code></pre>
<p>OR：</p>
<pre><code>
wget https://github.com/skschoo1/skphp/archive/v2.0.zip
unzip v2.0.zip
cd skphp-2.0
</code></pre>
<p>Next：</p>
<pre><code>
composer update
</code></pre>

# Shortcut
<pre><code>
composer create-project --prefer-dist skschool/skphp skphp
</code></pre>
